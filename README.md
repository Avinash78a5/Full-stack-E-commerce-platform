Full Stack E-Commerce Platform

A modern Full Stack E-Commerce web application built using **React.js**, **Node.js**, **Express.js**, **Supabase**, **Redux Toolkit**, and **Stripe**. The application provides a complete shopping experience with secure authentication, category-based product browsing, cart management, online payments, and order management.

---

## 📌 Project Overview

This project is inspired by modern e-commerce platforms like **Flipkart** and **Amazon**. It demonstrates the implementation of a complete shopping workflow, from user authentication to successful payment and order placement.

The application focuses on clean architecture, reusable components, scalable routing, secure payment integration, and efficient state management.
